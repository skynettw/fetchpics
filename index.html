    Welcome to Fetchpics

    這是一個簡單的Python程式，主要的功能就是提供給它一個網頁，然後它會把這個網頁中所有超過50K的圖形檔儲存在pics資料夾中。由於不同的網頁有不同的編排方式，所以如果你給的網頁沒有找出任何的檔案，那麼表示你還要再觀察一下網頁的原始碼，改一下使用的下載標籤才行。

    使用方法

    Python fetchpics targetURL

    注意事項

    這個程式使用的是 Python 2.7，需要安裝 requests 和 BeautifulSoup4 模組才行
